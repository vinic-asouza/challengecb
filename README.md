# Challenge

Aplicação desenvolvida em React.

# FRONTEND

# Tecnologias utilizadas:

- React
- HTML
- CSS
- Javascript
- Gulp 
- Stylus 
- Metodoliga CSS (BEM)

# Intruções para inicialização do projeto:

    Instalar dependências:

    $ npm i

    Rodar projeto:

    $ npm start

    Para fazer alterações no CSS, é necessário rodar o comando do gulp em outro terminal:

    $ npm run gulp
